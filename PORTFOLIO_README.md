# Gargi Gautam - Portfolio Website

A modern, dark-themed portfolio website showcasing professional work, skills, and projects.

## 🎨 Features

- ✨ **Dark Theme** - Beautiful dark color scheme with gradient accents
- 🎬 **Smooth Animations** - Fade-in, slide, and scroll animations
- 📱 **Fully Responsive** - Works perfectly on desktop, tablet, and mobile
- ⚡ **Performance Optimized** - Fast loading and smooth interactions
- 🎯 **Interactive Elements** - Hover effects, ripple effects, cursor glow
- 📋 **Complete Sections** - Hero, About, Skills, Projects, Experience, Contact
- 🔗 **Social Integration** - Links to GitHub and LinkedIn

## 🚀 Quick Start

### Local Testing
1. Download all files
2. Open `portfolio.html` in your browser
3. That's it! No installation needed.

### Deploy on Vercel

1. Push files to GitHub:
```bash
git init
git add .
git commit -m "Add portfolio"
git branch -M main
git remote add origin https://github.com/gargigautam511/portfolio
git push -u origin main
```

2. Go to https://vercel.com
3. Click "Import Project"
4. Select your GitHub repo
5. Click "Deploy"
6. Your portfolio is now live!

### Deploy on GitHub Pages

1. Create a new GitHub repo named `username.github.io`
2. Push your portfolio files
3. Visit `https://username.github.io`

## 🎨 Customization

### Change Colors
Edit the CSS variables in `portfolio-styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #f093fb;
    --dark-bg: #0f0f1e;
    --darker-bg: #0a0a14;
    --light-text: #e0e0e0;
    --lighter-text: #ffffff;
    --border-color: #2a2a3e;
    --success-color: #00d4ff;
}
```

### Update Social Links
Edit the HTML social links in `portfolio.html`:
```html
<a href="https://github.com/YOUR-USERNAME" target="_blank">
<a href="https://www.linkedin.com/in/YOUR-PROFILE/" target="_blank">
```

### Add Projects
Add new project cards in the Projects section:
```html
<div class="project-card">
    <!-- Project content -->
</div>
```

### Edit Content
Simply edit the text in `portfolio.html` directly.
